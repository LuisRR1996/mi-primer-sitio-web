# Mi Primera Web Site 🚀

Este es mi primer proyecto de maquetación web estática, desarrollado como parte del curso de Fundamentos de Programación Web en SENATI. El proyecto partió como un ejercicio de clase, al cual le implementé personalizaciones de interfaz y estilos propios.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructuración semántica del sitio (`<header>`, `<nav>`, `<main>`, `<section>`).
* **CSS3:** Hoja de estilos externa con personalización de caja (margin, padding, border-radius) y uso de Flexbox para la distribución de contenedores.
* **Bootstrap 5:** Framework (vía CDN) para agilizar el diseño responsivo, la creación de tarjetas (cards) y la estructura del formulario.
